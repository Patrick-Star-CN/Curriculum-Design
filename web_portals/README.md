# web_portals
  
大二下前端课大作业

一个简简单单的用前端三件套魔改了 hexo 主题的简单信息分享站，借用了沈老师的一些博客，万分感谢。

## 环境与启动方式

无需额外环境，用 vscode 的 live server 插件即可运行。

## 项目结构

```
.
├── 2023                          // 文章归纳目录
├── 404.html                      // 找不到网页后显示的目录 
├── README.md                     // 介绍文档
├── about                         // “联系我”网页的目录
├── categories                    // “分类”网页的目录
├── css                           // css 目录
│   ├── assets                    // 一些字体布局有关的 css 文件目录
│   ├── fontawesome               // 字体设置相关 css 目录 
│   └── layout                    // 布局相关 css 目录
├── fonts                         // 字体目录   
├── images                        // 图片目录
├── index.html                    // 主页面
├── js                            // js 脚本目录
│   ├── layouts                   // 布局相关 js 目录
│   ├── libs                      // 引用第三方库 js 目录
│   ├── plugins                   // 选用插件目录
|   └── tools                     // 工具类目录
├── links                         // 友情链接目录
├── login                         // “登录”网页的目录
├── music                         // 音乐播放器的数据源文件目录
└── page                          // 分页相关的目录
```
